Documentation for TILE Interface

ActiveBox

TILE_ENGINE

Constructor: TILE_ENGINE({attach:(optional){String},toolSet:{Array}})

Arguments:
attach - {String} what to attach the interface to (defaults to "body")
toolSet - {Array} Array of plugin objects that are fed into the TILE_ENGINE to be used as plugins
These are displayed in the "Tools" drop-down

loads in html layout from columns.json, parses JSON and sends to all
other objects

Functions
getBase - checks TILECONSTANTS.json for base http data
setUp - makes Transcript, ActiveBox, sets global binds
addNewTool - sets a new tool - users must define tool in .js code and proper JSON format
saveSettings - starts prompt to save the current manifest of data as a JSON object
clear_Image_Tags - clears the log bar of all tags for a particular image
clear_All_Tags - clears the LogBar completely
